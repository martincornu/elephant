# E-lephant

E-lephant est une appliation de e-learning interactive qui révolutionne la formation en entreprise. Ce repo héberge un prototype de l'application sur l'OS Android.

## Fiche technique

Avec 6 mois supplémentaire l'application pourrait être dans l'environnement suivant :

![Alt text](martincornu/elephant/blob/dev/app/src/main/res/drawable/ecosysteme.jpg?raw=true "Eco-système")

- Heroku : pour déployer et exécuter l’application. Avantages : conteneurisation, mise à l’échelle.
- API REST Django
- Serveur Gunicorn/nginx
- SGBD PostgreSQL : robuste, communauté présente et configuré par le PaaS Heroku.
- Cloud AWS3 : capacité de stockage à la demande.
